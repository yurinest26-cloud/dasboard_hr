# HR Dashboard — Проектная работа

Интерактивный дашборд для анализа HR-метрик: численности, зарплат, стажа, удовлетворённости.

Цель
Создать удобный инструмент для HR и линейных руководителей, позволяющий:
- Анализировать структуру
- Выявлять выбросы и риски
- Принимать решения на основе данных

Технологии
- Python
- Streamlit
- Plotly
- Pandas
- Excel (2 листа)

Установить зависимости из Requirements.txt

Как запустить?
Запустите из командной строки:
streamlit run hr_dashboard.py
